# 42 CPP Copilot

This extension improve your experience in the CPP Pool. It helps you to implement automatically canonical class 

## Install
Launch Quick Open with ⌘+P and enter
`ext install bbird21.42cpp-copilot`

## Usage
Launch Quick Open with ⌘+P and enter
`cppCopilot`
